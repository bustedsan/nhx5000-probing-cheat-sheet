Probe Power & Activation (DMG Mori Manual)

🔌M144 / M145 — Probe ON/OFF

( ACTIVATE SPINDLE PROBE ) M144 ( Probe Power ON — transmitter activated ) G4 P1.0 ( Dwell 1 sec — allow probe to initialize ) ( ... probing cycle ... ) ( DEACTIVATE SPINDLE PROBE ) M145 ( Probe Power OFF )
M144 activates the spindle probe transmitter. Always dwell after M144 before calling any G320/G370 cycle. M145 deactivates. These are DMG Mori NHX5000 manual-specified M-codes.

🔧Probe Orientation (Required)

( ORIENT SPINDLE FOR PROBE USE ) M19 ( Spindle orient — align probe stylus ) M144 ( Probe ON ) G4 P1.0 ( Wait for probe ready ) G91 G28 Z0 ( Return to ref Z )
Do NOT rotate spindle (S-code) while probe is installed. M19 orient only.

⚙Probe Setup Program Template

G320/G370 cycles here ... ) M145 ( Probe OFF ) G91 G28 Z0 ( Z home ) M30 ( Program end )

G320 — Automatic Measurement Cycle (DMG Mori Native)

▶G320 Syntax — Full Format

G320 X_ Y_ Z_ D_ F_ W_ Q_ L_ S_ ( X = Target X position ) ( Y = Target Y position ) ( Z = Target Z position ) ( D = Measurement direction 1=+X 2=-X 3=+Y 4=-Y 5=+Z 6=-Z ) ( F = Feed rate (in./min) ) ( W = Work offset number to write result e.g. W54 = G54, W55 = G55, etc. ) ( Q = Axis to write offset (1=X 2=Y 3=Z) ) ( L = Measurement distance (search stroke) ) ( S = Skip signal selection )

📈G320 — Measure X Origin (Single Surface)

( SET G54 X-ZERO FROM PART EDGE ) M144 ( Probe ON ) G4 P1.0 G90 G0 ( Absolute rapid ) G0 X-1.0 Y0. Z0.5 ( Approach safe position ) G320 X0. Y0. Z0.5 D1 F10. W54 Q1 L0.5 ( G320 probes in +X direction, writes result to G54 X offset automatically via W54 Q1 ) M145 ( Probe OFF )

📈G320 — Measure Y Origin (Single Surface)

( SET G54 Y-ZERO FROM PART FRONT FACE ) M144 G4 P1.0 G0 X0. Y-1.0 Z0.5 G320 X0. Y0. Z0.5 D3 F10. W54 Q2 L0.5 ( D3 = probe in +Y direction W54 Q2 = write to G54 Y offset ) M145

📈G320 — Measure Z Surface

( SET G54 Z-ZERO FROM TOP OF PART ) M144 G4 P1.0 G0 X0. Y0. Z1.0 G320 X0. Y0. Z0. D6 F5. W54 Q3 L0.5 ( D6 = probe in -Z direction W54 Q3 = write to G54 Z offset ) M145

📈G320 — Boss / Center X (Two Touches)

( FIND CENTER OF BOSS IN X ) ( Touch -X side first ) M144 G4 P1.0 G0 X-1.25 Y0. Z-0.5 G320 D1 F10. L1.0 #100=#5001 ( Save +X touch in #100 ) ( Touch +X side ) G0 X1.25 G320 D2 F10. L1.0 #101=#5001 ( Save -X touch in #101 ) #102=[#100+#101]/2 ( Calc center ) G10 L2 P1 X#102 ( Write center to G54 X ) M145
#5001 = last measurement result variable (Fanuc Macro B).
G370 — Reference Face Measurement (DMG Mori Native)

▶G370 Syntax

G370 X_ Y_ Z_ D_ F_ W_ Q_ L_ ( G370: measures from a reference surface. Used to update offsets relative to a known reference (master) position. D = direction same as G320: 1=+X 2=-X 3=+Y 4=-Y 5=+Z 6=-Z W = work offset to update Q = axis (1=X 2=Y 3=Z) )
G370 is used when you need to re-reference to a datum surface rather than set an absolute zero. The cycle measures the surface and updates the offset relative to the reference point stored from the G323 sensor offset calibration.

📈G370 — Re-Reference Z from Datum Surface

( RE-REFERENCE Z FROM FIXTURE DATUM PAD ) M144 G4 P1.0 G0 X0. Y0. Z1.0 G370 X0. Y0. Z0.1 D6 F5. W54 Q3 L0.5 ( Probes down -Z, updates G54 Z offset relative to calibrated sensor reference ) M145

📈G370 — Re-Reference X from Datum

( RE-REFERENCE X FROM FIXTURE SIDE PAD ) M144 G4 P1.0 G0 X-1.0 Y0. Z0. G370 X-0.5 Y0. Z0. D1 F10. W54 Q1 L0.5 ( Probes +X to datum, updates G54 X ) M145

G323 — Sensor Offset Calibration (DMG Mori Native)

▶G323 Syntax

G323 F_ L_ ( G323: Calibrates spindle probe tip offset using the table-mounted reference tool sensor (TLS/reference sphere). F = probing feed rate L = search stroke distance Result stored in machine parameters / probe offset register )
Run G323 whenever the probe is changed, stylus is swapped, or after any crash. This establishes the probe tip position relative to machine coordinates. Must be run before G320/G370 cycles produce accurate results.

📈G323 — Probe Calibration Routine

( PROBE CALIBRATION — RUN AFTER PROBE INSTALL ) O0999 G90 G20 T50 M6 ( Load probe ) M19 ( Spindle orient ) M144 ( Probe ON ) G4 P1.0 ( Position over table reference sensor ) G0 X[table_sensor_X] Y[table_sensor_Y] Z1.0 G323 F5. L0.5 ( Stores probe offset in machine register ) M145 ( Probe OFF ) G91 G28 Z0 M30

⚠G323 — When to Run

ALWAYS run G323 calibration after:
• Installing or re-installing the probe
• Replacing probe stylus
• Any probe crash or over-travel
• Machine startup / power-on sequence
• Any change to probe tool offset (T50)
Store the G323 calibration run as a sub-program and call it as part of your setup macro before any G320/G370 measurement cycles.
